# User List App (React Native)

## Features

* Fetch users from API
* Display users in card format
* Search users by name
* Click on user to view details
* Loading indicator and error handling

## Tech Stack

* React Native (Expo)
* Axios
* Expo Router

## API Used

https://jsonplaceholder.typicode.com/users

## How to Run

```bash
npm install
npx expo start
```
